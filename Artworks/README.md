# Notice -注意

Windose Installer reads the configuration file corresponding to each artwork (which has the same name as the illustration file itself, except the extension is different, which is .ini). The format is as follows:

Windose Installer 会读取每个插画所对应的配置文件（与插画文件本身同名，但后缀名为 ini）。格式如下：

# Configuration format - 配置格式

[WindoseInstaller]

ArtworkInfo="Here goes the information of artwork author"
